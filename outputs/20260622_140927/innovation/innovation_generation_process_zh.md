# 创新点生成过程

## 用户问题

做功能连接分析的创新性实验

## 方法概述

1. LLM 根据用户问题、文献和候选范式生成多条候选创新点。
2. 每条候选创新点生成后进行自我反思，检查新颖性、可证伪性、缺失数据依赖和风险控制。
3. 自我反思后不再做统一评审，而是采用 pairwise tournament：每次比较两个 idea，记录胜者、败者和中文理由。
4. 将所有胜负关系输入 Bradley-Terry-Luce 排序；优先使用 `choix.ilsr_pairwise`，失败时使用本地 BTL 排序。
5. 实验/计算执行完成后，根据 execution_results 反向更新每个创新点；如果没有执行结果，也会记录“尚未被结果检验”的更新状态。

## innovation_02：Graph-based dynamic analysis

- 创新点：构建一个时变的图网络模型，通过计算跨时间窗口的全局模块化系数（Q）和关键网络组（DMN-TPN）连接权重，量化认知负荷诱导的网络拓扑结构重组（Topological Reorganization）机制。
- 科学假设：高认知负荷状态下，大脑网络会从模块化、局部化的状态重组为高度整合的、全局连接增强的状态，具体表现为DMN-TPN连接权重显著增加，但全局模块化系数显著下降。反之，低认知负荷状态下，网络倾向于维持高模块化和局部连接。
- 提出依据：现有研究（[papers[0]]）证实了功能连接的动态性，但多仅关注连接强度（Edge Weight）的变化。图论（[papers[1]]）提供了模块化和效率等指标来量化网络的组织原则。本方案将动态性（[papers[0]]）与拓扑结构（[papers[1]]）结合，旨在填补‘认知负荷驱动的网络拓扑重组机制’这一关键空白。
- 新颖性说明：现有工作要么停留在静态FC分析（仅关注平均连接强度），要么仅关注时间序列的连接变化（如[papers[0]]的简单时变FC）。本方案的创新在于：1) 不仅跟踪连接权重，而是计算每个时间窗的全局拓扑指标（如$Q$），提供了一个系统性的‘网络组织原则’的量化描述；2) 将认知负荷（行为指标）作为驱动变量，实现对网络重组机制的因果性探究，超出了单纯的关联分析。
